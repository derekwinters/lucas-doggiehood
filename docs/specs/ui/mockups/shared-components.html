PillButton

Decoration/gift choices, dialogue actions, menu buttons. One shape, tinted by role. Press collapses the shadow and drops the button down.

Dialogue box

DialogueBox

The shared panel shell for conversation (#175) and onboarding (#176) surfaces — name tag, body, action row. Screens fill the content; the chrome is defined once here.

ConstantValueRegion

Regions: name tag (overlapping tab) · body text · action row. The specific actions per screen (e.g. whether a decline button exists) are decided in that screen’s wireframe, not here.
